# Markdown Extended Syntax

Table	  \| Syntax | Description |
        | ----------- | ----------- |
        | Header | Title |
        | Paragraph | Text |
        
       
Fenced Code Block	      \```
                        \{
                           "firstName": "John",
                           "lastName": "Smith",
                           "age": 25
                         }
                        ```
                        
Footnote	    Here's a sentence with a footnote. [^1]

              [^1]: This is the footnote.
